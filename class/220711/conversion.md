### 자료형 변환

- 파이썬에서 데이터 형태는 서로 변환할 수 있음
- 암시적 형 변환

  - 사용자가 의도하지 않고, 파이썬 내부적으로 자료형을 변환

```python
True + 3 # 4
3 + 5.0 # 8.0
3 + 4j + 5 # 8+4j
```



- 명시적 형 변환

  - 사용자가 특정 함수를 활용하여 의도적으로 자료형을 변환

  - 형식만 일치한다면 바꿀 수 있다.


  - int, float, str (형식에 맞는 문자열만 가능)

```python
# 문자열은 암시적 타입 변환이 되지 않음
'3' + 4 # => TypeError

# 명시적 타입 변환이 필요함
int('3') + 4 # => 7

# 정수 형식이 아닌 경우 타입 변환할 수 없음
int('3.5') + 5 # => ValueError

float('3.5') + 5 # 8.5
```
